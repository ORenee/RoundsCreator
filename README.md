
BROKEN THINGS - DO NOT USE
 -
- Manually adding couples
- Latin Dances/Rounds
- Standard Dances/Rounds
- Mambo
- Viennese Waltz

HOW TO USE
-
- Format your data like this sheet: https://docs.google.com/spreadsheets/d/1mrVGqt0mK9YGe99VaQFSQTeb8MgB8wVB1hHlVre9vUg/edit?usp=sharing
- Export data as a .csv
- Copy and paste the contents into the data/database.txt file
- Run App
