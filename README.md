
BROKEN THINGS - DO NOT USE
 -
- Manually adding couples
- Latin Dances/Rounds
- Standard Dances/Rounds
- Mambo
- Viennese Waltz

HOW TO USE
-
- Format your data like this sheet: https://docs.google.com/spreadsheets/d/1mrVGqt0mK9YGe99VaQFSQTeb8MgB8wVB1hHlVre9vUg/edit?usp=sharing
- Export data as a .csv
- Open the .csv in a program that is not excel so it's actually in a csv format and not a spreadsheet
- Copy and past the contents (WITHOUT THE TITLES) into the data/database.txt file
- Run App
